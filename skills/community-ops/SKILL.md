# community-ops：社群运营诊断与规划

> 💡 配合 wechat cli 直接读取微信聊天记录使用，效果更佳

## 一、开始前必问（诊断输入）

收集以下信息再给建议：

1. **社群目标**：引流？成交？交付？留存？
2. **产品/服务**：卖什么，客单价多少？
3. **现有资源**：好友数量、是否有KOL合作、是否有老学员/铁杆粉丝
4. **社群现状**：准备新建 or 已有群（几人、活跃度如何）？
5. **遇到的具体问题**：不活跃 / 不成交 / 裂变失败 / 不知道怎么开始？

---

## 二、社群六大类型 & 选型诊断

| 类型 | 核心目标 | 适用阶段 | 关键动作 |
|------|---------|---------|---------|
| **引流群** | 从公域获取精准用户 | 冷启动/拉新 | 福利钩子+每日输出+信任积累 |
| **粉丝群** | 维护铁杆，提升复购 | 有一定用户规模后 | 专属感+持续价值+优先购 |
| **快闪群** | 短期集中爆发成交 | 新品发售/限时活动 | 3天SOP+裂变+制造稀缺 |
| **训练营群** | 完成学习交付+转化后端 | 有课程产品 | 三峰值体验+结营转化 |
| **交付群** | 服务履约，提升完课率 | 产品售出后 | 打卡+答疑+督学机制 |
| **铁杆粉丝群** | 核心用户运营，口碑裂变 | 老用户沉淀 | 高度参与感+共创+优先权 |

**选型公式**：
- 没用户 → 引流群优先
- 有用户没成交 → 快闪群/训练营
- 有成交想复购 → 粉丝群/铁杆粉丝群
- 有课要交付 → 训练营群+交付群

## 三、建群前置条件检查

运营社群前必须满足：

**基础门槛**
- [ ] 微信好友 ≥ 500（建议1000+才做快闪群）
- [ ] 有明确的产品/服务可以变现
- [ ] 朋友圈内容正常输出（≥3个月记录）

**资源准备**
- [ ] KOL资源（有1-3个有影响力的合作方）
- [ ] KOC资源（忠实老用户，能主动分享）
- [ ] 志愿者/内部托（社群启动时烘托氛围）

**内容准备**
- [ ] 入群欢迎语 + 群规
- [ ] 钩子福利（实物/资料/课程/优惠）
- [ ] 3-7天的内容排期

> ⚠️ 如果前置条件不满足，优先补齐资源，不要强行启动

## 四、快闪群完整SOP（3天发售打法）

### 启动前（D-3 至 D-1）

**物料清单**
- [ ] 产品海报（9张以上，横/竖版各备）
- [ ] 群主/KOL个人故事文案
- [ ] 产品价值说明文案
- [ ] 成交截图/学员见证
- [ ] 钩子福利设计（进群即得）
- [ ] 快速回复话术库

**渠道激活顺序**
1. 先激活KOL（提前1-3天沟通，确认参与）
2. 向老用户/铁杆粉丝发私信邀请
3. 公域（朋友圈/视频号/公众号）预热发酵
4. 裂变机制启动（邀请X人得额外福利）

**裂变设计原则**
- 钩子要「即时可得」（不要等群满了才发）
- 设置进度可见（群人数/裂变排行）
- 24小时内兑现承诺

### Day 1：讲故事（建信任）

时间节点：晚上8-10点（黄金时段）

内容结构：
```
群主故事（为什么做这件事）
→ 学员见证（成果展示）
→ 福利预告（明天公布产品）
→ 互动话题（引发共鸣）
```

运营动作：
- 欢迎新成员（@每个人）
- 志愿者/KOC提前埋点互动
- 处理退群/静默用户（私信唤醒）

### Day 2：讲方法（建价值）

内容结构：
```
干货分享（解决用户痛点）
→ 方法论拆解（让用户感到「有用」）
→ 产品引出（这套方法我系统化了）
→ 限时优惠预告
```

运营动作：
- 答疑互动（增加粘性）
- 截图传播引导（"截图发朋友圈得XXX"）
- 私信高意向用户

### Day 3：成交日

**16字发售口诀**：发起接龙 → 捡大便宜 → 提前预定 → 制造稀缺

成交时间轴：
```
20:00 开场预热（倒计时）
20:30 正式发售（价格/权益公布）
21:00 截图见证（成交截图轰炸）
21:30 稀缺催促（"还剩X个名额"）
22:00 最后冲刺（截止前30分钟）
22:30 收网/感谢
```

话术要点：
- 接龙格式：「1. 姓名+已付款」（制造从众）
- 截图要显示真实付款记录
- 稀缺必须是真实的（不能虚假限量）

### 发售后

- 24小时内私信未成交用户（追销话术）
- 发送录音/回放给错过直播的人
- 7天内复盘：转化率、GMV、退款率、裂变系数

## 五、训练营社群设计框架

### 定位：前端引流 × 后端转化

训练营本身是低价/免费产品，核心目的是：
1. 完成价值交付（提升信任）
2. 在结营时转化高客单后端产品

### 短训营 vs 长训营

| 维度 | 短训营（3-7天） | 长训营（21-30天） |
|------|--------------|----------------|
| 适合产品 | 技巧类/工具类 | 习惯养成/系统学习 |
| 完课率 | 较高 | 较低（需督学机制） |
| 转化时机 | 第3-5天结营 | 第7天+结营双转化 |
| 运营成本 | 低 | 高 |

### 三个峰值体验（必须设计）

**1. 破冰峰值（第1天）**
- 自我介绍模板（降低门槛）
- 助教@每个人欢迎
- 即时送出小礼物/资料

**2. 超额价值峰值（中期）**
- 超出预期的内容/嘉宾
- 学员成果展示（激励完成）
- 专属答疑/1v1诊断机会

**3. 结营峰值（最后一天）**
- 结营仪式感（证书/纪念品）
- 成果汇报/优秀学员表彰
- 后端产品发售（情绪最高点转化）

### 三角色团队配置

| 角色 | 职责 |
|------|------|
| 主讲（IP） | 内容输出、信任背书 |
| 助教 | 督学、答疑、私信跟进 |
| 班长/志愿者 | 氛围烘托、带动打卡 |

> 1个助教建议负责≤50人

### 训练营10要素规划

1. 训练营名称（吸引人的成果描述）
2. 学员画像（精准定位目标人群）
3. 核心承诺（学完能得到什么）
4. 课程模块（3-7个主题）
5. 每日任务设计（可完成+有成就感）
6. 打卡机制（格式模板+审核标准）
7. 激励体系（积分/排行/结业礼）
8. 助教分工（谁负责哪些学员）
9. 后端产品（结营转化什么）
10. 复盘指标（完课率/转化率/NPS）

### 用户完整链路

```
公域流量
  → 低价/免费训练营（信任积累）
    → 结营发售中端产品（1:1诊断/小班课）
      → 高端产品（私教/高阶社群）
        → 铁杆粉丝（代言人/合伙人）
```

## 六、社群活跃度维护

### 日常运营节奏（适用于粉丝群/引流群）

| 时间 | 动作 |
|------|------|
| 早8-9点 | 早安话题/打卡 |
| 午12点 | 干货小卡片/tip |
| 晚8-10点 | 主要内容输出/互动 |
| 随时 | 回复提问，@表扬活跃用户 |

### 周度节奏

- 周一：本周预告（制造期待）
- 周三：干货分享（核心价值日）
- 周五：福利/优惠（转化日）
- 周日：总结/答疑（留存日）

### 活跃度异常处理

| 症状 | 可能原因 | 解决方案 |
|------|---------|---------|
| 发消息没人回 | 内容不够吸引/成员不精准 | 发红包破冰 + 提问引互动 |
| 大量退群 | 消息太多/内容太泛 | 减少广告，增加干货比例 |
| 只看不说话 | 参与门槛高 | 设计简单互动（投票/接龙/晒图） |
| 突然没人发消息 | 话题断档 | 助教/托抛话题，@几个活跃用户 |

## 八、常见问题快速诊断

**Q：群刚建起来没人说话怎么办？**
→ 托/志愿者提前埋好互动话题，欢迎每个新人，第一天必须红包破冰

**Q：快闪群裂变没效果怎么办？**
→ 检查：钩子是否即时兑现？邀请流程是否简单（1步完成）？海报是否有吸引力？

**Q：训练营完课率低怎么办？**
→ 每日任务是否太难？助教是否有私信跟进？有没有督学激励（掉队公示/追回机制）？

**Q：结营转化率低怎么办？**
→ 检查三峰值是否都设计到位？发售时机是否在峰值情绪最高点？价格锚点是否合理？

**Q：社群做了半年没变现怎么办？**
→ 先确认是否有明确的变现产品，再看是否从未做过集中发售；建议做一次3天快闪群测试

## 九、重要原则

1. **先有产品再建群**：没有产品的社群是情感消耗，不是商业行为
2. **精准 > 数量**：100个精准用户比10000个泛用户更有价值
3. **承诺必须兑现**：裂变钩子、训练营承诺、优惠截止时间，一旦说出去必须做到
4. **高频输出建信任**：社群冷了就难复温，保持每天至少一条有价值的内容
5. **转化要有仪式感**：发售不是简单发个链接，需要铺垫+情绪+稀缺+行动指令
